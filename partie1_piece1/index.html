    PARTIE 1 PIÈCE 1¶

    I. PRESENTATION DE LA PIECE¶

    • Nom de la pièce : Pièce 1  
    • Système d’unité : MMGS (millimètre, gramme, seconde)  
    • Matériau : Acier AISI 1020  
    • Densité : 0,0079 g/mm³  
    • Tolérance : ± 1 %  
    • Décimales : 2 (toutes les dimensions seront arrondies à 2 décimales)  

    Paramètres géométriques : - A = 81.00 mm - B = 57.00 mm - C = 43.00 mm

    Description :   Cette pièce est l’élément de base de l’étude.   La pièce 1 est une pièce mécanique simple, conçue selon les dimensions spécifiées par les paramètres A, B, et C.   Elle comporte des trous débouchants, conformes aux spécifications, et est destinée à être modélisée en respectant strictement les cotes données.   Cette pièce permet notamment de valider la capacité à créer des modèles paramétrés précis ainsi que la maîtrise du calcul de masse avec un matériau donné.

    II. GAMME DE RÉALISATION DE LA PIÈCE 1¶

    1️⃣ Préparation du poste de travail¶

    • Vérifier que le logiciel SolidWorks est correctement configuré en système d’unités MMGS (millimètre, gramme, seconde).
    • S’assurer que le modèle est enregistré régulièrement sous des noms de fichiers distincts pour chaque étape (Piece1).

    2️⃣ Modélisation sous SolidWorks¶

    Étape 1 : Création de l’esquisse de base¶

    • Ouvrir une nouvelle pièce dans SolidWorks.
    • Dessiner l'esquisse a extruder selon les dimensions A = 81 mm, B = 57 mm, qui seront parametrer en utilisant equation et en respectant les cotes précises avec deux décimales.

    Étape 2 : Extrusion¶

    • Extruder l’esquisse sur C = 43 mm.

    Étape 3 : Vérification des dimensions et contraintes¶

    • Utiliser l’outil de cotation pour vérifier toutes les dimensions.
    • Vérifier la conformité des tolérances dimensionnelles.

    Étape 4 : Application du matériau¶

    • Attribuer à la pièce le matériau Acier AISI 1020 dans SolidWorks.

    Étape 5 : Calcul de la masse¶

    • Vérifier la masse calculée dans les propriétés de la pièce.

    Étape 6 : Contrôle qualité¶

    • Effectuer une revue des dimensions sur le modèle 3D.
    • Vérifier la cohérence des paramètres géométriques A, B et C.
    • Enregistrer la pièce finale sous un nom distinct pour la traçabilité.

    Étape 7 : Sauvegarde et documentation¶

    • Sauvegarder le fichier SolidWorks avec un nom clair et versionné.
    • Exporter les vues principales (isométrique, face avant) au format image pour inclusion dans la documentation.

    III. RÉALISATION¶

    Étape 1 : Création de l’esquisse de base¶

    Voici la liste des fonctions utilisées dans SolidWorks :

    Ligne (pour le contour de l’esquisse)  

    • Ouvrir une nouvelle esquisse sur la face .  
    • Cliquer sur l’icône Ligne dans la barre d’outils Esquisse.  
    • Cliquer une fois pour commencer la ligne, déplacer la souris, puis cliquer de nouveau pour la terminer.  
    • Répéter jusqu’à ce que le contour soit créé.

    Arc (pour le contour de l’esquisse)  

    • Choisir l’outil Arc (arc par 3 points ou arc tangent, selon besoin).  
    • Cliquer pour définir le premier point de l’arc, le deuxième, puis le troisième afin de former la courbe.

     

     

     

     

    Cercle (pour le cercle de diamètre 14 mm)  

    • Sélectionner l’outil Cercle.  

    • Cliquer afin de placer le centre, puis déplacer la souris et cliquer de nouveau afin de définir le rayon.   Cotation intelligente (pour appliquer les dimensions)  

    • Cliquer sur l’outil Cotation intelligente.  

    • Cliquer sur une ligne, un cercle ou entre deux entités afin de définir une cote.  
    • Déplacer la cote afin de la positionner et cliquer de nouveau.  
    • Taper la valeur numérique souhaitée et valider.  

    PARAMÉTRAGE DES ÉQUATIONS¶

    Parametrage de A, B et C. - Aller dans le menu Outils (Tools) en haut de l'écran.   - Cliquer sur Équations… (Equations).   - La boîte de dialogue « Équations, variables globales et cotes » s’ouvre. - Créer 3 variables globales : - Taper :   - A = 81   - B = 57   - C = 43 - Clique sur « Ajouter (Add) » après chaque afin de les valider.  

     

     

    Relations d’esquisse (pour assurer les contraintes)   - Sélectionner une ou plusieurs entités (lignes, points, cercles).   - Depuis la barre d’outils d’esquisse, appliquer la relation appropriée (parallel, perpendicular, coïncident, horizontal, vertical, tangente…).

    Étape 2 : Extrusion¶

    • Depuis l’onglet Fonctions (Features), cliquer sur Extruded Boss/Base (Bossage/Base Extrudée).  
    • SolidWorks sélectionne automatiquement l’esquisse active.  
    • Saisir la valeur de la profondeur d’extrusion (C = 44 mm) dans le panneau de gauche.  
    • Cliquer sur la coche verte afin de valider.  
    • La pièce 3D est générée à partir de l’esquisse.

     

     

    Étape 3 : Vérification des dimensions et contraintes¶

    • Ouvrir l’esquisse.  
    • Sélectionner l’esquisse à vérifier et l’ ouvrir en mode édition.  
    • Activer l’affichage des contraintes (icône « Afficher les relations d’esquisse ») ou via le menu contextuel.  
    • Observer les symboles de contraintes (perpendicular, parallel, coïncident...).  
    • Vérifier que toutes les entités sont noires (esquisse complètement définie).  
    • S’il y a des entités bleues, ajouter ou modifier les contraintes jusqu’à ce que l’esquisse soit complètement contrainte.

    Étape 4 : Application du matériau¶

    • Clic droit sur Matériau dans l’arbre de création (FeatureManager) à gauche.  
    • Sélectionner Modifier le matériau dans le menu contextuel.  
    • Choisir Acier AISI 1020 dans la bibliothèque des matériaux.  
    • Cliquer sur Appliquer, puis sur Fermer.  
    • Vérifier que le matériau est bien pris en compte.

     

     

    Étape 5 : Calcul de la masse¶

    • S’assurer que le matériau est appliqué.  
    • Aller dans l’onglet Évaluer en haut.  
    • Cliquer sur Propriétés de masse (Mass Properties).  
    • Consulter la masse affichée dans la fenêtre.

     

    Étape 6 : Contrôle qualité¶

    • Revue des dimensions sur le modèle 3D.  
    • Vérifier la cohérence des paramètres géométriques A, B et C.  
    • Enregistrer la pièce finale sous un nom distinct afin d’en assurer la traçabilité.

    Étape 7 : Sauvegarde et documentation¶

    • Sauvegarder le fichier SolidWorks avec un nom clair et versionné.   -Exporter les vues principales (isométrique, face avant) au format image afin de les inclure dans la documentation.
